Adds a chic and modernist "Abstract" section to posts via a shortcode [abstract].

== Description ==

The Abstract Box plugin allows you to easily add a stylish and modern "Abstract" section to your WordPress posts using a simple shortcode. Perfect for highlighting key takeaways, summaries, or any important content you wish to stand out.

== Installation ==

1. Upload the `abstract-box` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the [abstract] shortcode in your posts to add an "Abstract" section.

== Usage ==

Simply wrap your abstract content with the [abstract] shortcode like so:

[abstract]Your abstract content here.[/abstract]

You can also customize the title of the "Abstract" section using the 'title' attribute:

[abstract title="Key Insights"]Your customized abstract content here.[/abstract]
